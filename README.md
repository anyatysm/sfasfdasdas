# GitHub User Finder
Автор: Иван Иванов

## Описание
Графическое приложение для поиска пользователей GitHub по имени. Позволяет сохранять избранных пользователей в JSON-файл.

## Использование API
- Введите имя пользователя в поле.
- Нажмите "Поиск".
- В появившемся окне отображается информация.
- Чтобы добавить пользователя в избранное, нажмите кнопку "Добавить в избранное".
